# Travel Booking Application - Frontend

This is the frontend part of the Travel Booking Application, built using React. The application allows users to search for travel bookings, view details, and make reservations.

## Features

- User-friendly interface for searching and booking travel.
- Responsive design for optimal viewing on various devices.
- Integration with a backend API for handling bookings and user authentication.

## Getting Started

To get started with the frontend application, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd travel-booking-app/frontend
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```

   The application will be available at `http://localhost:3000`.

## Folder Structure

- `public/`: Contains the static files, including the main HTML file.
- `src/`: Contains the React components, pages, services, hooks, and styles.
  - `components/`: Reusable components like Header, Footer, and BookingForm.
  - `pages/`: Different pages of the application such as Home, SearchResults, and BookingDetails.
  - `services/`: API service for making requests to the backend.
  - `hooks/`: Custom hooks for managing state and side effects.
  - `styles/`: CSS files for styling the application.
